Responsive Tabs


Instructions:
  1. Resize your browser window to see how tabs scale / respond
  2. Add / remove tabs

Usage

Turns any Boostrap ".nav-tabs" container into responsive tabs:

  $('.tabs-container').responsiveTabs({
    minWidth: "100",
    maxWidth: "200"
  }
